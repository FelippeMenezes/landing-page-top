# landing-page-top
A landing page project created to practice HTML and CSS layout skills for The Odin Project's Foundations course.

## 🚀 Live Demo
[View the Live Preview here](https://felippemenezes.github.io/landing-page-top/)

## 🎯 Project Goal

The challenge consists of:

- Recreating a provided layout (reference image)
- Structuring HTML properly
- Styling using CSS
- Organizing elements with Flexbox
- Building a visually appealing and functional page

> The focus is not pixel-perfect accuracy, but strengthening layout and positioning fundamentals.

---

## 🧠 What I Learned

During the development of this project, I practiced:

- 📄 Semantic structure with HTML5  
- 🎨 Styling with CSS  
- 📦 Layout using Flexbox  
- 🔧 Code organization  
- 🧩 Separation of concerns (HTML vs CSS)  
- 💡 Solving layout and positioning challenges  

---

## 🏗️ Page Structure

The landing page is composed of:

1. **Header**
   - Logo
   - Navigation links

2. **Hero Section**
   - Main title
   - Subtext
   - Call-to-action button
   - Image

3. **Information Section**
   - Cards with images and descriptions

4. **Quote Section**
   - Highlighted quote or testimonial

5. **Call to Action**
   - User engagement section
   - Action button

6. **Footer**
   - Copyright information
